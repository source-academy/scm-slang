The language ``Scheme \S 3'' (scm-slang) is a teaching language for the SICP
courseware in Source Academy. It is broadly aligned with R5RS Scheme, but is
not a complete implementation of the standard and documents its own deviations
and library behavior.

This document specifies the Chapter 3 language, which is the default and full
Scheme language used in Source Academy for scm-slang.
