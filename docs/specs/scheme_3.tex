\section{Changes}

Compared to Scheme \S 2, Scheme \S 3 adds the following:
\begin{itemize}
  \item \texttt{set!} mutation
  \item \texttt{begin} sequencing
  \item \texttt{delay} and promise support
  \item Vector literals (\texttt{\#(...)} form)
\end{itemize}
